# FormField

<SingleComponentStatus frontmatter={props?.pageContext?.frontmatter} />

`FormField` and the suite of components outlined here are the foundation for the
to be developed, `TextField`, `SelectField`, and `TextareaField`.

FormField renders its children inside a context that provides ids and attributes
that enable these components to relate to each other according to WCAG guidelines.

##### Basic usage with floating label

```jsx live
() => {
  const [value, setValue] = useState('');
  return (
    <FormField>
      <FormControlDecoratorGroup
        leadingElement={<Icon src={FavoriteBorder} />}
        trailingElement={<Icon src={Verified} />}
        floatingLabel="What kind of cats?"
      >
        <FormFieldControl value={value} onChange={(e) => setValue(e.target.value)} />
      </FormControlDecoratorGroup>
      <FormFieldDescription>We answer the important questions here.</FormFieldDescription>
    </FormField>
  );
}
```

##### Input types

```jsx live
() => {
  const [value, setValue] = useState('');
  return (
    <>
      <FormField size="sm">
        <FormControlDecoratorGroup floatingLabel="What kind of cats?">
          <FormFieldControl
            leadingElement={<Icon src={FavoriteBorder} />}
            value={value}
            onChange={(e) => setValue(e.target.value)}
          />
        </FormControlDecoratorGroup>
      </FormField>
      <FormField>
        <FormControlDecoratorGroup floatingLabel="What kind of cats?">
          <FormFieldControl
            as="textarea"
            leadingElement={<Icon src={FavoriteBorder} />}
            value={value}
            onChange={(e) => setValue(e.target.value)}
          />
        </FormControlDecoratorGroup>
      </FormField>
      <FormField size="lg">
        <FormControlDecoratorGroup floatingLabel="What kind of cats?">
          <FormFieldControl
            as="select"
            leadingElement={<Icon src={FavoriteBorder} />}
            value={value}
            onChange={(e) => setValue(e.target.value)}
          >
            <option value="">Select a cat</option>
            <option>Persian cat</option>
            <option>British Shorthair</option>
            <option>Maine Coon</option>
            <option>Siamese cat</option>
          </FormFieldControl>
        </FormControlDecoratorGroup>
      </FormField>

  );
```
